# 智能家居控制系统使用文档

## 项目简介

这是一个基于Java Swing开发的智能家居控制系统图形界面应用程序。该系统允许用户通过直观的图形界面控制家中的智能设备，包括电视、灯具、洗衣机和空调等。系统采用现代简约的设计风格，提供友好的用户体验，并支持设备状态的持久化存储。

## 系统功能

### 核心功能
- **设备控制**：开启/关闭各类智能家电设备
- **状态显示**：实时显示各设备的开关状态
- **数据持久化**：自动保存和加载设备状态
- **单例控制器**：统一的设备控制中心

### 支持的设备类型
1. 📺 电视 (tv) - 小米电视
2. 💡 灯具 (lamp) - LED灯泡  
3. 🧺 洗衣机 (washma) - 美的洗衣机
4. ❄️ 空调 (air) - 美的空调

## 技术架构

### 设计模式
- **单例模式**：`smartname` 类作为设备控制器，确保全局只有一个控制实例
- **继承体系**：所有设备类都继承自 `jiadian` 基类
- **接口实现**：`Switch` 接口定义设备开关行为

### 主要组件

#### 1. 设备基类 (`jiadian.java`)
- 实现 `Switch` 接口，提供基本的开关功能
- 包含设备名称和状态属性
- 支持序列化，便于数据持久化

#### 2. 具体设备类
- `tv.java` - 电视设备类
- `lamp.java` - 灯具设备类  
- `washma.java` - 洗衣机设备类
- `air.java` - 空调设备类

#### 3. 控制器 (`smartname.java`)
- 单例模式实现
- 提供设备控制方法 `control()`
- 提供批量状态查询方法 `printallstatus()`

#### 4. 数据管理 (`DataManager.java`)
- 负责设备状态的保存和加载
- 使用对象序列化技术存储到 `smart_home_data.dat` 文件

#### 5. 图形界面 (`SmartHomeGUI.java`)
- 基于Swing构建现代化用户界面
- 采用卡片式布局设计
- 支持鼠标交互和视觉反馈

## 使用说明

### 启动应用
1. 运行程序后，将显示智能家居控制主界面
2. 界面顶部显示标题 "🏠 智能家居控制系统"
3. 中间区域以卡片形式展示4个智能设备
4. 底部提供操作按钮

### 设备控制
1. **查看状态**：每个设备卡片右侧显示当前状态（开启/关闭）
2. **控制设备**：
   - 点击设备对应的控制按钮可切换开关状态
   - 开启状态时按钮显示为红色"关闭"
   - 关闭状态时按钮显示为绿色"开启"
3. **状态反馈**：操作后会弹出提示框显示操作结果

### 功能按钮
- **🔄 刷新**：重新加载并显示设备状态
- **💾 保存**：手动保存当前设备状态到文件
- **🚪 退出**：退出程序并自动保存状态

### 数据持久化
- 程序启动时自动从 `smart_home_data.dat` 文件加载历史状态
- 如果文件不存在，则使用默认配置（电视和灯开启，洗衣机和空调关闭）
- 窗口关闭或点击退出按钮时自动保存当前状态
- 可随时点击"保存"按钮手动保存状态

## 扩展开发指南

### 添加新设备类型

1. 创建新的设备类，继承 `jiadian` 类：
```java
package demo;

public class refrigerator extends jiadian {
    public refrigerator(String name, boolean status) {
        super(name, status);
    }
    
    // 可以添加冰箱特有的方法
    public void setTemperature(int temp) {
        // 设置温度逻辑
    }
}
```

2. 在 `SmartHomeGUI.java` 中修改设备数组大小和初始化：
```java
devices = new jiadian[5]; // 增加数组大小
devices[4] = new refrigerator("海尔冰箱", false);
```

3. 更新界面循环次数和设备图标数组

### 自定义设备属性

可以在具体的设备类中添加特有属性和方法：
```java
public class smartLight extends lamp {
    private int brightness; // 亮度
    
    public void setBrightness(int level) {
        this.brightness = level;
    }
    
    public int getBrightness() {
        return this.brightness;
    }
}
```

### 修改UI样式

在 `SmartHomeGUI.java` 中可以调整：
- 颜色常量（PRIMARY_COLOR, SUCCESS_COLOR等）
- 字体大小和样式
- 窗口尺寸和布局参数
- 边框和内边距

## 常见问题

### Q1: 程序无法启动？
**A:** 检查是否安装了JDK，并确保环境变量配置正确。在命令行输入 `java -version` 验证。

### Q2: 设备状态没有保存？
**A:** 检查当前目录下是否有写入权限，`smart_home_data.dat` 文件应该与程序在同一目录。

### Q3: 如何恢复默认设置？
**A:** 删除 `smart_home_data.dat` 文件，重新启动程序即可使用默认配置。

### Q4: 界面显示异常？
**A:** 确保操作系统支持Java Swing，尝试更新JDK版本或检查显卡驱动。

## 注意事项

1. **线程安全**：Swing界面操作必须在事件调度线程(EDT)中执行
2. **序列化兼容**：修改设备类结构时注意保持 `serialVersionUID` 的一致性
3. **文件路径**：数据文件保存在程序运行目录，注意相对路径问题
4. **内存管理**：及时释放不需要的资源，避免内存泄漏
